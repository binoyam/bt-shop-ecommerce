# fullstack app
## bt-shop ecommerce web application
#### FRONTEND: REACT, BACKEND: FLASK, DATABASE: MYSQL

# INSTALLATION
### (frontend)
- git clone https://github.com/binoyam/bt-shop-ecommerce.git
- cd frontend
- npm install
- npm start
### (backend)
- cd ../backend
- pip install -r requirements.txt
- python app.py
